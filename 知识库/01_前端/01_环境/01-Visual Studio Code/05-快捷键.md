| 快捷键                  | 介绍                    |
| ----------------------- | ----------------------- |
| shift + alt + 下/上箭头 | 快速复制一行            |
| ctrl + d                | 选定多个相同的单词      |
| ctrl + h                | 替换                    |
| ctrl + shift + h        | 全局替换                |
| ctrl + g                | 快速定位到某一行        |
| ctrl + +/-              | 放大缩小整个编辑器界面  |
| ctrl + alt + 上/下箭头  | 添加多行光标            |
| ctrl + 左/右箭头        | 光标定位到单词首/单词尾 |
| shift + delete          | 删除光标所在行          |
| shift + alt + f         | 格式化                  |
| alt + shift             | 选择区块                |
| alt + 上/下箭头         | 上下移动选择行          |
